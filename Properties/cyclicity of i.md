
The powers of iii repeat in a cycle of four. If the exponent is a multiple of four, the result is one. If the exponent is one more than a multiple of four, the result is iii. If the exponent is two more than a multiple of four, the result is negative one. And if the exponent is three more than a multiple of four, the result is negative iii. Then the pattern starts over again.

$$
i^n= \begin{cases}1 & \text { if } n=4 k \\ i & \text { if } n=4 k+1 \\ -1 & \text { if } n=4 k+2 \\ -i & \text { if } n=4 k+3\end{cases}
$$
